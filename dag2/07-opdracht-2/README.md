# Opdracht 2

We gaan nu verschillende technieken combineren. In script.js zit al een stukje
code waarmee je URLs van afbeeldingen kunt downloaden. We maken voor elke URL
een img tag en schrijven deze naar het scherm.

Maak met CSS alle img tags 80px breed en hoog. Geef de img tags ook deze CSS:
object-fit: cover;
border-radius: 50%;

Nu heb je drie ronde afbeeldingen in je HTML. Voeg hier ook een <button> aan toe.
Je mag deze eventueel zelf CSS geven.

Hierna luister je met JavaScript naar een klik op de knop. Als er op
de knop geklikt wordt geef je elke image een andere transform style waardoor
deze naar een andere plek op het scherm verplaatst.

Een paar tips:
* Werk stap voor stap, je kan altijd hulp vragen als je vast zit.
* Werkt je CSS niet? Gebruik rechtermuisknop en "inspect" in de browser om te
  zien waarom.
* Werkt je JavaScript niet? Gebruik de console om te zien of er errors zijn.
